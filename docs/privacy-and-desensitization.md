# Privacy and Desensitization

This repository is public-safe by design. It contains synthetic data only, generic table aliases, adapter interfaces, mock/file adapters, scoring/backtesting engines, and tests.

Do not commit private hosts, credentials, internal table names, real store/anchor/product/SKU names, screenshots, logs, RPA/Hue dumps, raw CSV/XLSX/ZIP exports, or generated runtime outputs. The `.gitignore` is intentionally strict for these artifact classes.
